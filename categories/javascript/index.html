ImageScrapBook

Image Scrap Book is a Chrome extension from which a user can select an image on a site. Logging in to the service website will show the user's saved images, which can also be titled, tagged or deleted and shared with others. The user can also share images in link format e.g. 'http://myservice.com/i/aabc3'
