ImageScrapBook

Posted Dec 19, 2018

GitHub repo

OBJECTIVES

Image Scrap Book is a Chrome extension from which a user can select an image on a site. Logging in to the service website will show the user’s saved images, which can also be titled, tagged or deleted and shared with others. The user can also share images in link format e.g. “http://myservice.com/i/aabc3"

Images are one of the most prevalent forms of media on today’s Internet. However, it is often hard to organize and transfer images between devices. Though users often use built-in screenshot features to save images locally, accessing the images from other devices is not easy without the use of cloud storage.

Additionally, screenshotted images may not be at the original resolution, nor is it always easy to find the source images. Therefore, we propose a web app that utilizes browser extensions to allow images to be easily saved and shared with other people and devices

SKILLS

  • Chrome extension API
  • jQuery.js
  • Django
  • Docker
  • minio
